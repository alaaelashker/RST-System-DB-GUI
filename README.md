# RST-System-DB-GUI

## Description
A desktop-based University Management System developed as part of a Database Systems university project.

## Overview
This project demonstrates a database-driven system with a graphical user interface for managing academic data and workflows.

## Technologies
- Python
- Tkinter (GUI)
- SQL Server
- SQL

## Features
- Role-based system (Student, Doctor, Teaching Assistant, Dean)
- Database design with primary and foreign keys
- Organized academic data management
- GUI connected to a relational database

## Purpose
This project was developed to apply database concepts and strengthen practical understanding of SQL and GUI-based systems.
